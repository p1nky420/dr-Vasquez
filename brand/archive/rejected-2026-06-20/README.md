# Exploraciones rechazadas

Estos archivos se conservan únicamente como registro del proceso.

No deben:

- utilizarse en la web;
- presentarse como propuestas vigentes;
- incorporarse al brand book;
- exportarse como archivos finales;
- tomarse como base obligatoria para futuras soluciones.

Motivo del rechazo: las exploraciones dependían excesivamente de la misma estructura FV superpuesta y no alcanzaban el nivel de diferenciación, legibilidad ni propiedad requerido.
